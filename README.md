# Diplomatura Inteligencia Artificial

Proyecto de trabajo para la Diplomatura de IA, organizado por módulos y con enfoque práctico en creación de contenidos web interactivos.

## Objetivo
Desarrollar materiales y prototipos por módulo (M6-M9), apoyados en agentes especializados y documentación fuente, para acelerar preparación de clases y entregables.

## Alcance actual
- Módulos previstos: **M6, M7, M8, M9**
- Metodología de trabajo: **Obsidian + notas atómicas**
- Salida objetivo: **páginas web interactivas y responsivas**

## Estructura del repo
- `agentes/` → definición de agentes de apoyo
- `documentacion/` → PDFs y material fuente de módulos
- `modulos/M6/` → MOC, notas atómicas y prototipo web interactivo
- `memory/` → notas de seguimiento del proyecto

## Como guardar cambios en GitHub
Cada vez que quieras guardar tu trabajo en GitHub, usá estos 3 comandos:

```bash
git add .
git commit -m "descripcion corta del cambio"
git push
```

Ejemplo:

```bash
git add .
git commit -m "actualizo web del modulo 6"
git push
```

Si alguna vez querés ver si hay cambios sin guardar:

```bash
git status
```
