Proyecto de trabajo para la Diplomatura de IA, organizado por módulos y con enfoque práctico en creación de contenidos web interactivos.

Desarrollar materiales y prototipos por módulo (M6-M9), apoyados en agentes especializados y documentación fuente, para acelerar preparación de clases y entregables.
