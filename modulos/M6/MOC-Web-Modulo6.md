# MOC - Página Web Interactiva: ChatGPT para Entrevistas de Trabajo

**Visión general (Atomic Note):**
Esta web interactiva transforma el Módulo 6 en un segundo cerebro digital accesible online. Usuarios navegan por prompts, ejemplos y tácticas como en un grafo de Obsidian, con interactividad para personalizar CVs, bios y emails en tiempo real.

**Objetivo web:**
Crear un sitio responsive donde reclutadores/estudiantes en Latam practiquen preparación laboral con ChatGPT.
