# Preguntas Reflexivas

## Idea atómica
El módulo prioriza preguntas abiertas para respuestas profundas.

## Desarrollo
- Ejes: experiencia, técnica, metodologías, adaptabilidad, habilidades blandas.
- Incluye generación de mejores respuestas por eje.

## Prompt sugerido
```text
Genera 3 preguntas abiertas por eje (experiencia, técnica, metodologías, adaptabilidad, blandas) y sugiere respuestas modelo.
```

## Aplicación web interactiva
- Input esperado del usuario: perfil profesional, vacante objetivo y nivel de experiencia.
- Transformación IA: análisis + redacción personalizada con ChatGPT.
- Output esperado en UI responsive: texto editable, checklist y versión final exportable.
