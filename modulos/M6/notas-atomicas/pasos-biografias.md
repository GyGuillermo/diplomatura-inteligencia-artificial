# Pasos Biografías

## Idea atómica
Definir titular profesional + foco de especialidad.

## Desarrollo
- Agregar evidencia (experiencia, proyectos, certificaciones).
- Cerrar con objetivo y valor para la empresa objetivo.

## Prompt sugerido
```text
Enumera pasos clave para escribir biografías profesionales impactantes y adaptables a distintas vacantes.
```
