# Importancia Currículum

## Idea atómica
El CV es la primera interfaz con reclutadores y sistemas ATS.

## Desarrollo
- Debe mostrar logros, stack técnico y ajuste al puesto.
- La personalización por vacante aumenta tasa de entrevista.

## Prompt sugerido
```text
Resume por qué un CV orientado al puesto mejora las probabilidades de ser preseleccionado y qué errores evitar.
```

## Aplicación web interactiva
- Input esperado del usuario: perfil profesional, vacante objetivo y nivel de experiencia.
- Transformación IA: análisis + redacción personalizada con ChatGPT.
- Output esperado en UI responsive: texto editable, checklist y versión final exportable.
