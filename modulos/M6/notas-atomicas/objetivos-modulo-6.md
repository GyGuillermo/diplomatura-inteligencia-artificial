# Objetivos Módulo 6

## Idea atómica
Aplicar ChatGPT para acelerar preparación laboral.

## Desarrollo
- Generar CV, biografía, preparación de entrevista y emails profesionales.
- Adaptar outputs a vacantes concretas con enfoque ATS.

## Prompt sugerido
```text
Actúa como experto en RRHH y guía un plan de preparación laboral en 4 frentes: CV, biografía, entrevista y email al empleador.
```

## Aplicación web interactiva
- Input esperado del usuario: perfil profesional, vacante objetivo y nivel de experiencia.
- Transformación IA: análisis + redacción personalizada con ChatGPT.
- Output esperado en UI responsive: texto editable, checklist y versión final exportable.
