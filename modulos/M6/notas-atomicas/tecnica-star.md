# Técnica STAR

## Idea atómica
Marco para respuestas conductuales: Situación, Tarea, Acción, Resultado.

## Desarrollo
- Facilita claridad, foco y evidencia de impacto.
- Útil para experiencia profesional y resolución de problemas.

## Prompt sugerido
```text
Convierte mis experiencias en respuestas STAR para preguntas de entrevista técnica y conductual.
```

## Aplicación web interactiva
- Input esperado del usuario: perfil profesional, vacante objetivo y nivel de experiencia.
- Transformación IA: análisis + redacción personalizada con ChatGPT.
- Output esperado en UI responsive: texto editable, checklist y versión final exportable.
